# Analyse-de-dataset

Réaliser une études sur un dataset fourni par https://www.kaggle.com/datasets/ et proposer une interface streamlit interactive pour explorer les données. Le dataset est à valider avec l'intervenant.

Utilisation des fonctionnalités de pandas :
- groupby (0.5)
- sum (0.5), moyenne (0.5) , écart type (0.5) et value_counts (0.5)
- Dérivation de nouvelles colonnes à partir de colonnes existantes (0.5)
- apply ou map (0.5)
- assign (0.5)
- 3 Graphiques de types différents (courbe, histrogramme, etc.)(3)

Streamlit :

- Interface jolie et fonctionnelle (2)
- Interface interactive (2)
- Application déployée sur internet (2)

Présentation:

- Présentation et Démo fluide (1)
- Présentation des résultats sous forme d'une inforgraphie (peut être réalisée avec un outil de dessin) (2)
- Réponse aux questions du prof (3)
- Fonctionnalités supplémentaire ou intéressantes (1)

Il est autorisé d'utiliser l'IA
